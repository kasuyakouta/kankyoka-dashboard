⚙️ スプレッドシート連携設定

Google Apps Script のウェブアプリURLを設定すると、入力のたびに自動でスプレッドシートへ保存されます。

設定手順:
1. 添付の kankyoka_gas.gs をコピー
2. スプレッドシート → 拡張機能 → Apps Script に貼り付け
3. デプロイ → 新しいデプロイ（種類: ウェブアプリ）
4. 実行ユーザー: 自分、アクセス: 全員
5. 生成されたURLをここに貼り付け
